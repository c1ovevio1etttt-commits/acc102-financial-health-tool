# Interactive Corporate Leverage & Financial Health Comparator

## Project Overview
This is an interactive Streamlit web app that allows users to compare the financial leverage and health of multiple companies using historical balance sheet data (2012-2020).

**Target Users**: Retail investors and finance students who want to quickly understand company debt risk without complex calculations.

## How to Run
1. Clone or download this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Features
- Select 2-6 companies and a specific year
- Financial health comparison table
- Asset Liability Ratio and Debt-to-Equity bar charts
- Simple English analysis summary for regular investors
- Historical trend line chart showing changes over years
- CSV download function

## Data Source
[Financial data of 4400+ public companies](https://www.kaggle.com/datasets/qks1lver/financial-data-of-4400-public-companies)  
Access date: April 2026
